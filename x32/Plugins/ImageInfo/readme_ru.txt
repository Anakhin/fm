Плагин "Image Info" для Far Manager 3.0
***************************************

Плагин, позволяющий просматривать информацию об изображениях (EXIF/IPTC/XMP).
Для чтения мета-данных изображений используются стандартные средства GDI+
и библитека exiv2 (http://www.exiv2.org).

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Хозяйке на заметку:
