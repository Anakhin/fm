Плагин "SQLite DB Browser" для Far Manager 3.0
**********************************************

Плагин, позволяющий просматривать таблицы баз данных SQLite.

Префикс плагина для вызова из командной строки: "sqlitedb:".

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).
